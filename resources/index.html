Free Learning Resources

High-quality, genuinely free resources for learning full-stack web development.

Full Curricula

  • The Odin Project A project-driven, open-source curriculum covering the full stack from HTML to Node.js. Widely considered the best free path for beginners.
  • freeCodeCamp Structured certifications spanning responsive design, JavaScript, front-end libraries, APIs, data visualisation, and more. Earns you verifiable certificates.
  • Full Stack Open University of Helsinki's deep-dive course covering React, Node, GraphQL, TypeScript, and testing. Industry-respected and completely free.
  • CS50W (Harvard) Harvard's free Web Programming course using Python, JavaScript, and SQL. Rigorous, structured, and comes with a verifiable certificate.

References & Guides

  • MDN Web Docs The definitive reference for HTML, CSS, and JavaScript. If you bookmark one thing, make it this.
  • JavaScript.info A thorough, modern guide to JavaScript from first principles to advanced topics like prototypes, async, and modules. Exceptionally well-written.
  • Web Dev (Chrome) Google's structured learning paths on performance, accessibility, CSS, HTML, forms, and modern web APIs.

Visual & Interactive

  • Web Skills An interactive visual map of everything involved in web development, with links to learning resources for each topic. Great for understanding the landscape.
  • roadmap.sh Community-maintained learning roadmaps for front-end, back-end, full-stack, DevOps, and more. Helpful for knowing what to learn next.

Patterns & Architecture

  • Patterns.dev Design and rendering patterns for modern JavaScript and React. Covers everything from singleton and observer to streaming SSR and islands architecture.
